### Convertidor de Texto a Mayúsculas

Este programa proporciona una interfaz simple para convertir archivos de texto a mayúsculas de manera asíncrona. Facilita la tarea de transformar múltiples archivos de texto al mismo tiempo.

#### Funcionalidades principales:
1. **Selección de archivos**: El usuario puede seleccionar uno o varios archivos de texto utilizando el botón "Seleccionar archivos". Se mostrará un cuadro de diálogo de selección de archivos donde se pueden elegir los archivos deseados.

2. **Conversión a mayúsculas**: Una vez seleccionados los archivos, el usuario puede iniciar el proceso de conversión haciendo clic en el botón "Convertir archivos". El programa leerá cada archivo de texto, convertirá su contenido a mayúsculas y guardará el resultado en un nuevo archivo con "_mayus" agregado al nombre original.

3. **Manejo de errores**: El programa maneja posibles errores que puedan surgir durante el proceso de lectura y escritura de archivos. Si ocurre algún problema, se mostrará un mensaje de error indicando la causa del fallo.

#### Uso:
1. Ejecutar el programa.
2. Seleccionar uno o varios archivos de texto haciendo clic en el botón "Seleccionar archivos".
3. Hacer clic en el botón "Convertir archivos" para iniciar el proceso de conversión.
4. Esperar a que se complete el proceso. Se mostrarán mensajes de estado y errores según corresponda.

#### Requisitos:
- El programa está diseñado para ejecutarse en sistemas Windows.
- Los archivos de texto deben tener la extensión ".txt" para ser procesados correctamente.
